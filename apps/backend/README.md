# Goc Nha Minh Backend

NestJS API for the Goc Nha Minh Digital Ecosystem.

## Scripts

```bash
npm install
npm run build
npm run start:dev
npm run start:prod
```

## Health

Local base URL:

```txt
http://localhost:3000/api/v1
```

Health endpoints:

```txt
GET /health
GET /health/db
GET /health/storage
GET /health/auth
```

Follow the contracts in `../../docs` before adding production modules.
